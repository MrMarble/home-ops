# Cloud Services

External services that complement the self-hosted infrastructure.

| Service | Purpose | Cost |
|---------|---------|------|
| [GitHub](https://github.com) | Repository hosting, CI/CD | Free |
| [Cloudflare](https://cloudflare.com) | Domain, DNS, R2 storage, tunnel | Free |
| [1Password](https://1password.com) | Secrets storage via [External Secrets](external:https://external-secrets.io) | ~€50 / year |
| [Backblaze B2](https://www.backblaze.com/cloud-storage) | Off-site backups | ~€20 / year |
| [Nabu Casa](https://www.nabucasa.com) | Home Assistant voice processing + cloud | ~€75 / year |

**Total: ~€145 / year**
